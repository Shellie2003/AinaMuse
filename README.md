# AinaMuse 🎸🎹

A powerful, easy-to-use music theory and transposition calculator built entirely in Python using the [Flet](https://flet.dev/) UI framework. 

Designed for guitarists, keyboardists, and vocalists, **AinaDigit-Music** simplifies the process of changing song keys, finding the right capo position, and learning new chord fingerings on the fly.

## ✨ Key Features

*   🎸 **Guitar Capo Calculator**: Select the original song key and your preferred playing key to instantly find the correct fret to place your capo.
*   🎹 **Keyboard Transposer**: Automatically calculates the exact semitone shift (`+` or `-`) needed for digital pianos and synthesizers.
*   🎼 **Diatonic Progressions**: Displays the natural chord progression (Major, minor, diminished) for any selected target key.
*   🖐️ **Dynamic Fretboard Diagrams**: Tap on any chord in the progression to view accurate, dynamically generated guitar fingerings. Includes multiple variations and barre chord support!
*   🔄 **Chord Conversion**: Side-by-side translation of common chords from the original key to the new playing key.
*   💾 **Save Calculations**: Bookmark your setups (Original Key → Playing Key) for quick reference during gigs, rehearsals, or practice sessions.
*   🌙 **Modern UI**: A sleek, fully responsive dark-mode interface.

## 🛠️ Built With

*   **Python 3.14
*   **Flet (v0.83.0)** - A framework that enables building interactive multi-platform apps in Python based on Flutter.

## 🚀 How to Use

1. **Select the Original Key**: Choose the key the song is originally written in.
2. **Select the Playing Key**: Choose the key you actually want to play on your instrument.
3. **Read the Results**: The app instantly provides the Capo fret and Keyboard transpose values.
4. **Explore Chords**: Check the *Chord Conversions* table or tap on a chord in the *Progression* section to learn how to play it on the guitar.
